# newton_fractals

#### This project graphically displays the basins of attraction for polynomial roots using Newton's Method.
#### Different colors represent convergence to different roots and the shading represents how fast a initial guess converged. (lighter = faster, darker = slower)
#### The x-axis represents the real part of initial guess x0 and the y-axis represents the imaginary part of the intiial guess. Each pixel is x + yi
